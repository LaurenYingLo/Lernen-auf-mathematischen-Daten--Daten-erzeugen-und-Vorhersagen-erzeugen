# Lernen auf mathematischen Daten

This repository contains a Jupyter Notebook exploring learning and prediction on mathematically generated data using AI methods.

## Content
- Generation of mathematical data
- Training models on the generated data
- Evaluating prediction performance

## Context
This notebook was created as part of a learning exercise on applying artificial intelligence to mathematical problems.

## Usage
Open and run the notebook:

- `Mathematischer Fortschritt mit KI.ipynb`
